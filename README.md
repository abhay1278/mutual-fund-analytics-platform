# Mutual Fund Analytics Platform

## Overview

This project is part of the Bluestock Fintech Capstone Project. The objective is to build a Mutual Fund Analytics Platform that collects, validates, analyzes, and visualizes mutual fund data using Python, SQL, and Business Intelligence tools.


## Datasets Used

1. Fund Master
2. NAV History
3. AUM by Fund House
4. Monthly SIP Inflows
5. Category Inflows
6. Industry Folio Count
7. Scheme Performance
8. Investor Transactions
9. Portfolio Holdings
10. Benchmark Indices
